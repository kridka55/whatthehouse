# คู่มือระบบทีมช่างลงพื้นที่ (Technician App)

ระบบนี้เพิ่มเข้าไปใน **Apps Script เดิม** (ใช้ Google Sheet + Drive ชุดเดียวกับ Dashboard) ช่างเข้าผ่านมือถือ → เลือกงาน → ถ่ายรูป+คอมเมนต์ทีละจุด → กดออกรายงาน PDF และสถานะงานจะเชื่อมกลับไปที่ Dashboard อัตโนมัติ

## ติดตั้ง (ทำครั้งเดียว)

1. เปิด Apps Script (โปรเจกต์เดิม)
2. ไฟล์ **`Code.gs`** → ลบทั้งหมด → วาง `Code.gs` ตัวล่าสุด → 💾
3. กดปุ่ม **+** ข้าง Files → **HTML** → ตั้งชื่อ **`Technician`** (ตัว T ใหญ่ สะกดให้ตรง) → ลบของเดิม → วางเนื้อหา `Technician.html` → 💾
4. เลือกฟังก์ชัน **`setup`** → กด **เรียกใช้ (Run)** → อนุญาตสิทธิ์ (ครั้งนี้จะขอสิทธิ์เพิ่มสำหรับสร้างเอกสาร PDF → กด Allow)
5. **Deploy → Manage deployments → ✏️ → Version: New version → Deploy**

## เพิ่มรายชื่อช่าง

หลังรัน `setup` แล้ว ใน Google Sheet จะมีแท็บใหม่ชื่อ **"ช่าง"** — เพิ่มแถวละ 1 คน:

| ชื่อช่าง | PIN | ใช้งาน (yes/no) |
|---|---|---|
| สมชาย | 1234 | yes |
| สมหญิง | 5678 | yes |

- **ชื่อช่าง** = ชื่อที่ช่างใช้ล็อกอิน (ต้องตรงเป๊ะ)
- **PIN** = รหัสส่วนตัว (ตั้งเป็นตัวเลขก็ได้)
- **ใช้งาน** = `yes` เปิดใช้, `no` ปิดไม่ให้เข้า (เช่น ช่างลาออก)

## ลิงก์สำหรับช่าง

```
<Web app URL>/exec?app=tech
```
(เอา Web app URL เดิม มาต่อ `?app=tech`) — ส่งให้ช่าง บุ๊กมาร์กไว้บนมือถือได้เลย
ระบบจะจำการล็อกอินไว้ ครั้งต่อไปเปิดมาเข้าเลยไม่ต้องพิมพ์ใหม่

## ขั้นตอนใช้งานของช่าง

1. เปิดลิงก์ → ใส่ชื่อ + PIN → เข้าระบบ
2. เห็น **รายการงานที่ต้องตรวจ** (งานที่ยังไม่ส่ง) → แตะเลือกงาน
   - พอเลือก สถานะงานใน Dashboard จะขยับเป็น **"ตรวจหน้างาน"** อัตโนมัติ
3. **เพิ่มจุดที่ตรวจพบ**: เลือกพื้นที่/ห้อง → ลำดับ → คำอธิบาย → ถ่ายรูป (รูปถูกบีบอัดและเก็บเข้าโฟลเดอร์ Drive ของลูกค้ารายนั้น) → บันทึก ทำซ้ำได้เรื่อย ๆ
4. กด **สร้างรายงาน PDF** → ระบบรวมทุกจุดเป็นรายงาน (หัวรายงาน+ข้อมูลลูกค้า / รายละเอียดทุกจุดพร้อมรูป / สรุปภาพรวม / ช่องลายเซ็น) เก็บไฟล์ใน Drive และเปิดให้ดูทันที
   - สถานะงานใน Dashboard ขยับเป็น **"จัดทำรายงาน"** อัตโนมัติ
5. งานที่เหลือ (ส่งอีเมล + ปิดงาน) แอดมินทำต่อจากหน้า Dashboard เหมือนเดิม

## เชื่อมกับ Dashboard อย่างไร

- ช่างเริ่มตรวจ → สถานะ **ตรวจหน้างาน**
- ช่างออกรายงาน → สถานะ **จัดทำรายงาน** + ไฟล์ PDF อยู่ในโฟลเดอร์ลูกค้า
- แอดมินเห็นความคืบหน้าทั้งหมดในหน้า Dashboard (รวมการ์ดสรุป/Analytics) และกดส่งอีเมลปิดงานได้
