# คู่มือติดตั้งระบบจองคิว + Dashboard งานตรวจบ้าน

ระบบนี้ทำให้: ลูกค้ากรอกฟอร์มจองบนเว็บ → ข้อมูลเข้า **Google Sheet** อัตโนมัติ + สร้างโฟลเดอร์ **Google Drive** ชื่อ "ชื่อลูกค้า_โครงการ" + ส่ง **อีเมลแจ้งคุณ** ทันที จากนั้นคุณเปิด **หน้า Dashboard** เพื่อดูสถานะงาน เลื่อน Progress 5 ขั้น อัปไฟล์เข้า Drive และกดส่งอีเมลแจ้งลูกค้าเมื่อเสร็จ

ทำตามทีละขั้น ใช้เวลาประมาณ 10 นาที ทำครั้งเดียวจบ

---

## ขั้นที่ 1 — เปิด Google Apps Script

1. เปิด Google Sheet ของคุณ: https://docs.google.com/spreadsheets/d/1Lsn0A5U500QKYRVRh1WWlAACCfZe6lUukVAQmbqlrSQ/edit
2. ที่เมนูด้านบน กด **ส่วนขยาย (Extensions)** → **Apps Script**
3. จะเปิดหน้าต่างใหม่ขึ้นมา มีไฟล์ชื่อ `Code.gs` พร้อมโค้ดตัวอย่างอยู่

> การเริ่มจากในชีตแบบนี้ ทำให้สคริปต์ผูกกับบัญชี Google ของคุณ และเข้าถึงชีต/ไดรฟ์/อีเมลได้เลย

---

## ขั้นที่ 2 — วางโค้ด Code.gs

1. ในหน้า Apps Script **ลบโค้ดเดิมทั้งหมด** ในไฟล์ `Code.gs` ออก
2. เปิดไฟล์ `Code.gs` (ที่ผมสร้างให้) คัดลอกทั้งหมด แล้ว **วางทับ**
3. กดบันทึก (รูปแผ่นดิสก์ 💾 หรือ Ctrl+S)

**ตรวจค่าตั้งต้น** ในส่วน `var CONFIG` ด้านบนของโค้ด:

| ค่า | ใส่อะไร |
|---|---|
| `ADMIN_EMAIL` | อีเมลที่คุณอยากให้รับแจ้งเตือนเมื่อมีจองใหม่ (ตอนนี้ตั้งไว้ `krid.kt@gmail.com`) |
| `DASHBOARD_KEY` | **กุญแจลับ** สำหรับเปิดหน้า Dashboard — เปลี่ยนเป็นรหัสของคุณเอง เช่น `wth-suanluang-77` (อย่าให้คนอื่นรู้) |
| `SHEET_ID` / `DRIVE_PARENT_FOLDER_ID` | ใส่ให้แล้ว ไม่ต้องแก้ |

---

## ขั้นที่ 3 — เพิ่มไฟล์ Dashboard.html

1. ที่แถบซ้าย ข้างคำว่า **Files** กดปุ่ม **+** → เลือก **HTML**
2. ตั้งชื่อไฟล์ว่า **`Dashboard`** (พิมพ์แค่นี้ ระบบเติม `.html` ให้เอง — ต้องสะกดตรงนี้ให้ตรงเป๊ะ ตัว D ใหญ่)
3. ลบโค้ดตัวอย่างในไฟล์นั้นออก แล้ววางเนื้อหาจากไฟล์ `Dashboard.html` (ที่ผมสร้างให้) ลงไป
4. กดบันทึก 💾

ตอนนี้จะมี 2 ไฟล์: `Code.gs` และ `Dashboard.html`

---

## ขั้นที่ 4 — อนุญาตสิทธิ์ (ทำครั้งแรกครั้งเดียว)

1. ด้านบน ช่องเลือกฟังก์ชัน เลือก **`setup`** แล้วกด **Run (▶ เรียกใช้)**
2. จะมีหน้าต่างขออนุญาตเด้งขึ้น → กด **Review permissions / ตรวจสอบสิทธิ์**
3. เลือกบัญชี Google ของคุณ
4. ถ้าเจอหน้าจอ "Google hasn't verified this app" ให้กด **Advanced / ขั้นสูง** → **Go to … (unsafe)** — ปลอดภัย เพราะเป็นสคริปต์ของคุณเอง
5. กด **Allow / อนุญาต**

ถ้าสำเร็จจะไม่มี error สีแดง ถือว่าเชื่อมต่อชีตและไดรฟ์เรียบร้อย

---

## ขั้นที่ 5 — Deploy เป็นเว็บแอป (Web App)

1. มุมขวาบน กด **Deploy / ทำให้ใช้งานได้** → **New deployment / การทำให้ใช้งานได้ใหม่**
2. กดรูปเฟือง ⚙️ ข้าง "Select type" → เลือก **Web app**
3. ตั้งค่าดังนี้:
   - **Description:** อะไรก็ได้ เช่น `WTH booking v1`
   - **Execute as:** **Me (อีเมลคุณ)**
   - **Who has access:** **Anyone / ทุกคน** ← สำคัญ (เพื่อให้ฟอร์มบนเว็บส่งข้อมูลเข้ามาได้)
4. กด **Deploy** → อนุญาตสิทธิ์อีกครั้งถ้าถาม
5. คัดลอก **Web app URL** ที่ได้ (ลงท้ายด้วย `/exec`) — เก็บไว้ใช้ขั้นถัดไป

> ⚠️ ถ้าแก้โค้ดในอนาคต ต้องกด **Deploy → Manage deployments → ✏️ แก้ไข → Version: New version → Deploy** เพื่อให้เวอร์ชันใหม่มีผล (URL เดิมไม่เปลี่ยน)

---

## ขั้นที่ 6 — ต่อ URL เข้ากับฟอร์มบนเว็บ

**ส่ง Web app URL (`…/exec`) มาให้ผมในแชต** แล้วผมจะใส่ให้ในเว็บและอัปขึ้นให้เลย

หรือถ้าอยากทำเอง: เปิดไฟล์ `assets/js/main.js` หาบรรทัด
```js
var BOOKING_ENDPOINT = "PASTE_APPS_SCRIPT_URL_HERE";
```
แล้วแทนที่ `PASTE_APPS_SCRIPT_URL_HERE` ด้วย URL ของคุณ จากนั้นอัปไฟล์ขึ้น GitHub

---

## ขั้นที่ 7 — เปิดหน้า Dashboard

เปิดลิงก์นี้ (เอา Web app URL ของคุณมาต่อ `?key=` ตามด้วยกุญแจลับที่ตั้งไว้):

```
https://script.google.com/…/exec?key=wth-cnx-2026
```

> เปลี่ยน `wth-cnx-2026` เป็นค่า `DASHBOARD_KEY` ที่คุณตั้งในขั้นที่ 2
> **บุ๊กมาร์กลิงก์นี้ไว้** จะได้เปิดง่าย ๆ ทุกครั้ง และอย่าเผยแพร่ลิงก์นี้ให้คนอื่น

---

## หน้า Dashboard ใช้งานยังไง

- **แถบเหลืองด้านบน** = แจ้งเตือนจำนวนงานจองใหม่ที่ยังไม่ได้เริ่ม
- **การ์ดแต่ละใบ** = งานของลูกค้า 1 ราย แสดงชื่อ/เบอร์/อีเมล/โครงการ
- **แถบ Progress 5 ขั้น** = กดที่ขั้นไหน เพื่อตั้งสถานะงานเป็นขั้นนั้น (จองคิว → นัดหมาย → ตรวจหน้างาน → จัดทำรายงาน → ส่งรายงาน)
- **📁 เปิดโฟลเดอร์ Drive** = เปิดโฟลเดอร์ของลูกค้ารายนั้นเพื่ออัปโหลดรูป/รายงาน
- **✉️ ส่งอีเมลแจ้งลูกค้าว่าเสร็จแล้ว** = ส่งอีเมลพร้อมลิงก์โฟลเดอร์ให้ลูกค้า และตั้งสถานะเป็นขั้นสุดท้ายอัตโนมัติ

---

## ทดสอบให้ครบวงจร

1. เข้าเว็บหน้า **ติดต่อเรา** กรอกฟอร์มจอง (ใส่อีเมลทดสอบของคุณ) แล้วกดส่ง
2. เช็กว่า: มีแถวใหม่ใน Google Sheet ✓ มีโฟลเดอร์ใหม่ใน Drive ✓ มีอีเมลแจ้งเข้าอีเมลแอดมิน ✓
3. เปิด Dashboard จะเห็นงานใหม่ขึ้นแถบเหลือง ลองกดเลื่อน Progress และกดส่งอีเมลแจ้งลูกค้า
